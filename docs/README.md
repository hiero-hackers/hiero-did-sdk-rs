# Docs

This folder contains focused guides for using this SDK.

## Guides

- [`api-reference.md`](./api-reference.md): Public API reference across all crates with examples.
- [`create-did.md`](./create-did.md): Creation-only guide for `did:hedera` (all supported creation paths).
- [`testing.md`](./testing.md): Test strategy, commands, environment setup, and troubleshooting.

## Quick Start

1. Read `create-did.md` if your goal is creating DIDs.
2. Read `api-reference.md` for module-by-module function usage.
3. Read `testing.md` before running networked integration tests.
